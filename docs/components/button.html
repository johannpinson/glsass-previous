Buttons

Buttons include simple button styles for actions in different types and sizes.

Add the c-button class to <a>, <input> or <button> elements for a default button. There are classes c-button--primary and c-button--link for predefined primary and link buttons. 

<button class="c-button">default button</button>
<button class="c-button c-button--primary">primary button</button>
<button class="c-button c-button--link">link button</button>

<button class="c-button c-button--white">white button</button>

Shapes variants

By default, buttons have rounded edges, but it's possible to keep straight edges by changing the $c-button-round-default Sass variable. If this variable is set to false, it will automatically generate an additional class. 

<button class="c-button c-button--primary c-button--rounded">rounded button</button>
<button class="c-button c-button--primary c-button--pill">pill button</button>
<button class="c-button c-button--primary c-button--block">block button</button>

Button group

Create a stak of button and ajust border and margins.

<div class="c-button--group">
  <button class="c-button c-button--danger">Previous</button>
  <button class="c-button c-button--success">Confirm</button>
</div>

<div class="c-button--group">
  <button class="c-button c-button--primary">Left</button>
  <button class="c-button c-button--primary">Middle</button>
  <button class="c-button c-button--primary">Right</button>
</div>

<div class="c-button--group c-button--group-vertical">
  <button class="c-button c-button--light">Top</button>
  <button class="c-button c-button--dark">Bottom</button>
</div>

Sass references

Variables

Name Type Default value Description
$c-button-border-width Number unit $g-border-width Thickness of the border
$c-button-round-default Boolean true Allow to disable the default border-radius settings
$c-button-radius Number unit $g-border-radius Border-radius for buttons
$c-button-font-size Number unit $g-font-size-base-rem Font-size of the content
$c-button-font-ratio Number 1/8 The ratio to calculate smaller and larger size
$c-button-font-size-small Number unit Calculated from the ratio and the font size Font size for the small size
$c-button-font-size-large Number unit Calculated from the ratio and the font size Font size for the large size
$c-button-line-height Number $g-font-line-height-base Line-height for the content
$c-button-focus-shadow-width Number unit $g-shadow-width Size of the shadow used in some states
$c-button-padding-y Number unit .75rem - $c-button-border-width Vertical (block) padding
$c-button-padding-x Number unit .75rem - $c-button-border-width Horizontal (inline) padding
$c-button-padding-x-small Number unit Calculated from the ratio and the padding Horizontal padding for the small size
$c-button-padding-x-large Number unit Calculated from the ratio and the padding Horizontal padding for the large size
$c-button-padding-y-small Number unit Calculated from the ratio and the padding Vertical padding for the small size
$c-button-padding-y-large Number unit Calculated from the ratio and the padding Vertical padding for the large size
$c-button-variants List Generic theme colors Allow to add colors variants based on existing color name palette
