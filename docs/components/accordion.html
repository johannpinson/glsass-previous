Accordion

Accordions toggle the visibility of content.

For avoiding to add a script, the open/close behavior is direclty managed from the <details> element.
By adding the open attribute on the main element, the component will be opened by default.

Sass references

Variables

Name Type Default value Description
$c-accordion-icon Icon iv(select) Indicator of open/close status
$c-accordion-icon-size Number unit 1rem Size of the indicator icon
$c-accordion-focus-shadow-width Number unit $g-shadow-width Size of the shadow box
$c-accordion-focus-shadow-color Color rgba(cv(primary), 0.25) Color of the shadow box
$c-accordion-header-radius Number unit $g-border-radius Border-radius of header, used for focus state
$c-accordion-with-open-animation Boolean true Enable animation on opening
$c-accordion-animation-initial-gap Number unit -1rem Initial gap for the moving transition
(change it to 0 to keep only the opacity transition)
