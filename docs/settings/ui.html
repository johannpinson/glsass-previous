UI

Reminder: the settings.ui file is forwarded from the index file present in its folder.
To override the variables, you must import it separatly, as indicated in the overview section.

The ui file is similar to the other settings files, but it's only about aesthetics settings.

Sass references

Variables - General

Name Type Default value Description
$g-border-width Number unit 1px The default thickness of borders
$g-border-color Color cv("gray", 300) The default color of borders
$g-border-radius Number unit 0.25rem The default border-radius
$g-shadow-width Number unit 1px The default size of the shadow box
