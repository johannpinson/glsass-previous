Core

The core file regroups the main variables used through all the codebase. All of them are overridable when you import the file/folder.

You will find inside settings for fonts, spacers, responsive breakpoints, etc.

// Keep the default settings
@use "~glsass/src/settings.core";

// Or add your own settings
@use "~glsass/src/settings.core" with (
  $g-color-primary: #0000ff, // blue
  $g-color-secondary: #800080, // purple
);

Fonts

Sass references

Variables

Name Type Default value Description
$g-fonts-google List / String An empty list The Google font name and his style needed
$g-font-family-map List An empty list The name or the main font
$g-font-size-base Number unit 16px The default font-size in pixel
$g-font-size-base-rem Number unit 1rem The default font-size in rem
$g-font-line-height-base Number unitless 1.5 The default line-height

Spacing

Sass references

Variables

Name Type Default value Description
$g-spacer Number unit equal to 24px The default space calculated from the font-size and line-height
$g-spacers Map Calculated from the $g-spacer The five spaces "sizes" from 1 to 5
$g-spacers-negative Map Calculated from $g-spacers The spacers sizes with negative values.

Heading

Sass references

Variables

Name Type Default value Description
$g-headings-font-size-h* Number unit Depending of the hn level The font-size of headings, from h1 to h6 variables
$g-headings-margin-bottom Number unit $g-spacer / 2 The margin-bottom of headings
$g-headings-color Color null The default color
$g-headings-font-family String null The default font-family
$g-headings-font-weight String 700 The default weight (set to 700 or bold)
$g-headings-font-style String null The default font-style (italic)
$g-headings-styles Map Predefined to respect the vertical rhythm A map with font-size and line-height for each hn

Grid

Sass references

Variables

Name Type Default value Description
$g-grid-breakpoints Map 7 breakpoints, from sm to xl The default space calculated from the font-size and line-height
$g-grid-container-max-widths Map Depeding of $g-grid-breakpoints The max-width of the container depending on the breakpoint
$g-grid-columns Number 12 The number of columnns in the grid
$g-grid-gutter-width Number unit 1rem The gutter size (the space between each column)
