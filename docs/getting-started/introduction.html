Introduction

Glsass (spell it simply Glass) is a Sass framework based on ITCSS for extensible and responsive front development.
The framework provides generics/starter rules, a grid system, extensible components and utilities classes, with a global project architecture.

Since Glsass is focused on a SCSS environment, there is not download archive, no CDN or compiled CSS file.

It's up to you to build you own Sass project by using only what you need!
